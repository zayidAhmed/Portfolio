PaletteForge — Docs

What is PaletteForge?

A browser-based color palette generator. Pick a harmony type, set how many columns you want, and hit Generate — or press Space. Colors are derived from color theory so they always look good together.

Harmony Types

TypeWhat it does
AnalogousColors close together on the wheel — smooth, natural feel.
MonochromaticOne hue, varied lightness — minimal and cohesive.
ComplementaryOpposite hues — bold, high contrast.
TriadicThree hues 120° apart — vibrant and balanced.
Split-CompOne base + two neighbors of its complement — softer than Complementary.
FreeFully random hues — no rules.

Controls

ControlWhat it does
− / + (Columns)Change the number of swatches (3–8).
← Prev / Next →Navigate your generation history.
♥ SavedOpen saved palettes to load or delete them.
⚡ GenerateGenerate a new palette (locked colors are kept).

Keyboard Shortcuts

KeyAction
SpaceGenerate a new palette.
←Previous palette in history.
→Next palette in history.

Shortcuts work when the page body is focused (not when typing in an input).

Column Actions

Hover a color column (or tap on mobile) to reveal two buttons in the top-right corner:

🔒 LockPins this color — it won't change on the next Generate. Click again to unlock.
♡ SaveSaves the whole current palette to your Saved list.

Click any HEX, RGB, or HSL value to copy it to clipboard.

Exporting

FormatOutput
CSS VarsA :root { } block with HEX, RGB, and HSL custom properties.
JSONAn array of color objects with all values and names.
TextA plain .txt file with values listed line by line.

Saved Palettes

Palettes are stored in localStorage and persist across page reloads in the same browser. From the Saved panel you can Load any palette back into the editor or ✕ delete it.
